User Routes

# Route method Endpoint Params Required * Description
1 Signup POST /user/signup {"email": "user@example.com", "username": "abcde", password: "123456"} email, username, password For user registration on the platform
2 Login POST /user/login {"email": "user@example.com", password: "123456"} email, password Sign into user account
3 Recover Password POST /user/recoverpassword {"email": "user@example.com"} email For user's account password recovery; OTP will be sent to email
4 Reset Password POST /user/resetpassword {"email": "user@example.com", "code": "12345678", "new_password": "123456"} email, code, new_password Reset password. Use OTP sent to email from /recoverpassword
5 Update Profile PATCH /user/updateprofile/{user_id} {"full_name": "Jack Robinson", "birthday": "12-12-1912", "city": "awka", "sex": "male", "bio": "i code", "image_url": "url_or_path_to_profilepic"} full_name, birthday, city, sex Update user's profile
6 Follow other user(s) PATCH /user/follow/{user_id} {"follow_list": ["111111111111", "222222222222222"]} follow_list (must be an array) Follow other users on the platform
7 User Profile GET /user/{user_id} null null Fetch user's data


Hype Routes

# Route method Endpoint Params Required * Description
1 Hype POST /hype {"user_id": "1212121231121231", "hype": "A hype body here", img_urls: ["path_or_url_to_image1", "path_to_img2"]} user_id, hype To make a hype
2 Edit Hype PATCH /hype/edit/{hype_id} {"hype": "A hype body example being edited here", img_urls: ["path_or_url_to_image1", "path_or_url_to_image2"]} hype To edit a hype
3 Get user's hypes GET /hype/user/{user_id} null null Fetches all hypes made by user
4 Delete user's hypes DELETE /hype/delete/{hype_id} null null Deletes a hype made by user
